Twisted bilayer grapheneの電子状態計算

Bistritzer Macdonald model

ここでは、Twisted bilayer grapheneの電子状態を近似的にうまく記述できるモデルとして Bistritzer-Macdonald model（連続体近似などとも呼ばれる）を紹介しよう。Twisted bilayer grapheneの電子状態計算は近似なしでは非常に難しい。長周期構造のモアレ構造を作るため、単位胞の中に数多くの原子（魔法角1.1°付近で数万個の原子）が含まれるためである。また、長周期構造としてのモアレも完璧に周期的ではない（準周期的である）。こうした事情から直感的には、電子状態計算が非常に困難であるように思われるが、実は低エネルギー状態に注目する限りにおいては、問題をずっと簡単化できる。モアレの周期は元のグラフェンの周期に比べてはるかに大きい。これを反映して、波数空間ではモアレの波数は、グラフェンの波数に比べてはるかに小さくなる。このことから、モアレの層間の相互作用によって、グラフェンの異なるバレー（KとK’）は混ざり合わないので、グラフェンの各バレー周りで独立に有効模型を定義できる。つまり、元のグラフェンの詳細な構造はバレーの自由度に押し付け、一旦微細な構造は無視して良いということである。この簡単化で、各バレーの周りの小さなモアレ波数について行列要素を抽出すれば良い（長波長近似になっている）ことがわかる。詳しい導出は、以下の参考文献に任せることにして、実際にBistritzer Macdonald modelのHamiltonianを眺めてみよう。
